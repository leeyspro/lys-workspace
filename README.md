# NovaWorks 회사 소개 랜딩페이지

React + TypeScript + Vite로 만든 간단한 회사 소개 랜딩페이지입니다.

## 포함 섹션

- 헤더
- 메인 배너
- 서비스 소개
- 문의하기

## 실행 방법

의존성을 설치합니다.

```bash
npm install
```

개발 서버를 실행합니다.

```bash
npm run dev
```

브라우저에서 터미널에 표시되는 로컬 주소를 열어 확인합니다. 기본 주소는 보통 `http://localhost:5173`입니다.

## 빌드

배포용 파일을 생성합니다.

```bash
npm run build
```

빌드 결과를 미리 확인합니다.

```bash
npm run preview
```
