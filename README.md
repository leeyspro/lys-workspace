# Data Lab

ADsP, SQLD, AWS Cloud·AI 자격증 학습 자료를 정리하는 개인 학습 포털입니다.  
각 영역은 대시보드, 요약 노트, 문제유형, 학습 로드맵으로 빠르게 이동할 수 있도록 구성합니다.

## 주요 영역

- `ADsP`: 시험 정보, 과목별 핵심 요약, 문제유형, 학습 관리
- `SQLD`: 데이터 모델링, SQL 기본·활용, 실전 정리
- `AWS`: Certification Map, 인증 배지, 시험코드, 레벨별 상세 핵심 포인트

## 최근 반영

- AWS Study Map에 실제 인증 배지형 UI와 시험코드 연결
- 레벨 상세 페이지에 자격증별 배지, 코드, 핵심 범위 카드 추가
- AWS 왼쪽 메뉴를 여러 섹션 동시 펼침 방식으로 변경
- Data Lab 메인 화면을 학습 포털 구조로 개편

## 실행

```bash
npm install
npm run dev
```

개발 서버는 Next.js 기본 설정을 따릅니다. 보통 `http://localhost:3000`에서 확인합니다.

## 빌드

```bash
npm run build
```
